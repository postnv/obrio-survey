# Obrio Survey

[Deployed on Vercel](https://clario-login-form.vercel.app/)

## Installation:
- `fill .env variables`
```dotenv
SURVEY=astrology
NEXT_PUBLIC_URL=http://localhost:3000
```
- `npm i`
- `npm run dev`

## Technologies:

- Next.js
- Redux Toolkit + redux-persist
- linting + pre-commit hooks
